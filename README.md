# 📑 README — Binary Sentiment Analysis (긍정/부정)

📌 프로젝트 개요
	•	데이터셋: 감성대화말뭉치(최종데이터)_Training.xlsx
	•	목표: 대화 문장을 입력받아 긍정(기쁨) 또는 부정(분노/불안/상처/슬픔/당황) 으로 분류
	•	모델: Embedding → BiLSTM → Dense(sigmoid)
	•	출력: 입력 문장별 긍정 확률 + 최종 판정(긍정/부정)
